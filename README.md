# Квест на день рождения 🎉

Сайт-квест по QR-кодам. Именинник сканирует коды, спрятанные по этажам дома,
собирает мини-подарки и в финале находит тортик.

## Стек

Чистый HTML / CSS / JS, без сборки. Хостинг — GitHub Pages.

## Как настроить

1. Открой `config.js` — впиши имя, тексты этажей, подарки и подсказки.
2. Положи фото/видео в папку `media/` (см. `media/README.txt`).

## Как запустить локально

```
python3 -m http.server 8000
```

Открой http://localhost:8000

## Деплой и QR-коды

1. Залей файлы в репозиторий на GitHub.
2. Settings → Pages → Branch `main` / корень.
3. Открой `https://НИК.github.io/РЕПО/qr.html` — страница сама нарисует все
   QR-коды с правильным адресом. Распечатай, вырежи, спрячь по этажам.
   «Старт» отдай имениннику.

## Файлы

- `index.html` — сам квест (один экран на этаж, роутинг по `#1`…`#6`)
- `config.js` — всё наполнение (правишь только его)
- `app.js` — логика, прогресс, конфетти
- `style.css` — праздничный стиль
- `qr.html` — генерация QR для печати
- `media/` — фото, открытки, видео
