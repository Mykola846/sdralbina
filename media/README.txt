Сюда положи свои фото / открытки / видео-подсказки.

Имена файлов, которые ждёт config.js по умолчанию:

  floor1.jpg  — подсказка, где код 1 этажа (показывается на старте)
  floor2.jpg  — подсказка, где код 2 этажа (показывается на 1 этаже)
  floor3.jpg  — подсказка, где код 3 этажа
  floor4.jpg  — подсказка, где код 4 этажа
  floor5.jpg  — подсказка, где код 5 этажа
  floor6.jpg  — (не обязателен, 6 этаж по умолчанию без фото)
  cake.jpg    — фото тортика на финальной странице

Можно использовать любые имена и форматы (jpg/png/mp4) —
просто укажи правильный путь в config.js (поле hint.src).

Если файла нет — сайт не сломается: покажется только текстовая подсказка.
